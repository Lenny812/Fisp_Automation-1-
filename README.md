# FISP Report Generator

A web-based tool for generating NYC DOB-compliant Facade Inspection Safety Program (FISP/Local Law 11) reports.

##  About

FISP Report Generator streamlines the process of creating professional building facade inspection reports required by New York City's Department of Buildings. Built for Qualified Exterior Wall Inspectors (QEWIs) and engineering firms.

##  Features

- **Complete Building Information Form** - Address, block, lot, BIN, owner details
- **QEWI & Inspector Details** - License info, firm details, inspection team
- **Condition Tracking** - Safe, SWARMP, and Unsafe classifications
- **All DOB Appurtenances** - Fire escapes, parapets, balconies, railings, and more
- **Photo Uploads** - Up to 10 photos per condition, elevation photos, inspector photos
- **Company Logo** - Custom branding on reports
- **3 Report Formats** - Classic, Modern, and Detailed styles
- **DOB Compliant** - All required FISP sections included
